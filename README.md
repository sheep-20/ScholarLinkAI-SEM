# ScholarLink AI - Backend

这是 ScholarLink AI 项目的后端服务。该项目旨在通过大模型与向量检索技术，提供智能化的学术论文管理与推荐服务。

## 🛠 技术栈
* **框架:** Python 3.x + Flask + Flask-RESTX (Swagger UI)
* **数据库:** MySQL + SQLAlchemy ORM
* **配置管理:** YAML + 环境变量
